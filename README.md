# Customer Retention & Churn Analysis Dashboard

## 📌 Project Overview
This project analyzes customer churn using the Telco Customer Churn dataset. The dashboard provides insights into customer retention, churn patterns, revenue impact, and customer lifetime metrics.

## 🎯 Objectives
- Analyze customer churn trends
- Identify high-risk customer segments
- Understand customer lifetime patterns
- Provide business recommendations to improve retention

## 🛠 Tools Used
- Microsoft Power BI
- Power Query
- DAX

## 📊 Dashboard Pages
1. Executive Dashboard
2. Customer Retention Analysis
3. Customer Lifetime & Revenue Analysis
4. Business Insights & Recommendations

## 📈 Key Insights
- Churn rate is approximately 26.5%.
- Month-to-month contract customers churn the most.
- Fiber Optic customers have higher churn.
- Short-tenure customers are more likely to leave.
- Revenue loss is mainly due to churned customers.

## 💡 Business Recommendations
- Promote long-term contracts.
- Improve Fiber Optic customer support.
- Focus retention campaigns on new customers.
- Offer personalized discounts.
- Continuously monitor churn trends.
